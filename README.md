# Lab 13: Programming by Demonstration

## Installation

1. Install Node 25+
2. Clone this repo
3. Install dependencies with `npm install`
4. Start the application with `npm run dev`
5. Ensure
   the [Stretch Pose Server](https://github.com/robotics-capstone-26sp-team-1/catmull_ws/tree/main/src/stretch_pose_server)
   node is running and ROS bridge is too.
