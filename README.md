# Lab 13: Programming by Demonstration

## Installation

1. Install Node 24+
2. Clone this repo
3. Install PNPM
4. Install dependencies with `pnpm install`
5. Start the application with `pnpm dev`
6. Ensure
   the [AssistFour](https://github.com/robotics-capstone-26sp-team-1/catmull_ws/)
   node is running and ROS bridge is too.
