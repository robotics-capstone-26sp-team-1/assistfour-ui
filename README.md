# AssistFour UI

## Installation

1. Install Node 24+ (we used 26)
2. Clone this repo
3. Install PNPM (NPM works too)
4. Install dependencies with `pnpm install` (`npm install`)
5. Start the application with `pnpm dev` (`npm run dev`)
6. Ensure
   the [AssistFour](https://github.com/robotics-capstone-26sp-team-1/catmull_ws/)
   node is running and ROS bridge is too.
